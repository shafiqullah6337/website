# Entwicklung — Anleitung

Lokalen Dev‑Server starten:

Option A — Schnell (kein Node):

```bash
python3 -m http.server 8000
# Öffnen: http://localhost:8000
```

Option B — Mit Live‑Reload (empfohlen):

```bash
npm install
npm run dev
# Öffnen: http://localhost:5173
```

Hinweis: `npm install` installiert `live-server` basierend auf `package.json`.

Viel Erfolg beim Entwickeln! ✨
